game2048lib
===========

Library for playing 2048(http://gabrielecirulli.github.io/2048/) from python


Setup library
-------------

```
    $ pip install git+https://github.com/Zapix/game2048lib
```


Example of usage
----------------

```python
    >>> from game2048lib import Game2048
    >>> game = Game2048()
    >>> game.move_left()
    >>> game.score
    4
```
